# Security Policy

## Supported Versions

Only the latest major version of Econeura Cloud is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| 0.x     | :x:                |

## Reporting a Vulnerability

We make a best effort to ensure Econeura Cloud is secure and production-ready. 

If you discover a vulnerability, please **DO NOT** open a public issue. 
Instead, email our security team at **security@econeura.com**.

### Process
1.  We acknowledge receipt within 24 hours.
2.  We will provide an estimated timeline for the fix.
3.  Once fixed, we will issue a patch release.

## Security Features
Econeura Cloud implements:
- **RBAC** (Role Based Access Control)
- **Sanitized Inputs** (Zod Validation)
- **Secret Management** (Google Secret Manager)
- **Audit Logging** (Immutable Logs)

Thank you for helping keep Econeura secure.
